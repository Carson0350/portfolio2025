## User Story

**As a** visitor,  
**I want to** fill out a contact form,  
**so that** I can send Carson a message directly from the portfolio.

## Acceptance Criteria

- [ ] Contact form component created with React Hook Form
- [ ] Form fields: Name (required), Email (required), Message (required, textarea)
- [ ] Field labels and placeholders provide clear guidance
- [ ] Form uses shadcn/ui Input, Label, and Textarea components
- [ ] Submit button with loading state during submission
- [ ] Form is fully accessible (labels, ARIA attributes, keyboard navigation)
- [ ] Responsive layout with proper spacing
