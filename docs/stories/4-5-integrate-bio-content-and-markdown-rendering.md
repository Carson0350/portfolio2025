**Status:** Ready for Dev

## User Story

**As a** developer,  
**I want to** render markdown bio content with proper styling,  
**so that** the About section displays formatted text beautifully.

---

## Acceptance Criteria

- [ ] Markdown rendering library installed (react-markdown or similar)
- [ ] Bio content from `/content/bio/about.md` parsed and rendered
- [ ] Custom styling applied to markdown elements (paragraphs, links, lists)
- [ ] Links open in new tab with proper security attributes
- [ ] Typography components used for consistent styling
- [ ] Content updates without code changes (edit markdown file only)
- [ ] Error handling if bio content is missing or invalid
