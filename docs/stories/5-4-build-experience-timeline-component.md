## User Story

**As a** visitor,  
**I want to** see Carson's work history in a timeline format,  
**so that** I understand his career progression and experience level.

---

## Acceptance Criteria

- [ ] Experience section component created with Section and Container layout
- [ ] Section heading: "Professional Experience" or similar
- [ ] Timeline component displays jobs in reverse chronological order
- [ ] Each job shows: company name, role, dates, key achievements, technologies
- [ ] Visual timeline indicator (vertical line with dots/markers)
- [ ] Responsive layout (full timeline on desktop, simplified on mobile)
- [ ] Semantic HTML with proper heading hierarchy
