## User Story

**As a** developer,  
**I want to** define design system tokens (colors, typography, spacing, breakpoints),  
**so that** the portfolio has consistent visual styling across all components.

---

## Acceptance Criteria

- [ ] Tailwind config extended with custom color palette (primary, accent, neutral shades)
- [ ] Typography scale defined (font sizes, line heights, font weights)
- [ ] Spacing scale customized for portfolio needs
- [ ] Responsive breakpoints defined and documented
- [ ] Design tokens exported as TypeScript constants for programmatic access
- [ ] Documentation created in `/docs/design-system.md` explaining token usage
- [ ] Test page created demonstrating all design tokens
