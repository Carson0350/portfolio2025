**Priority:** P0 (Critical)  
**Status:** Ready for Dev  
**Assignee:** TBD

## User Story

**As a** visitor,  
**I want to** filter projects by technology or category,  
**so that** I can find projects relevant to my interests.

---

## Acceptance Criteria

- [ ] Filter buttons/tags added above projects grid (e.g., "All", "React", ".NET", "Azure")
- [ ] Clicking filter shows only projects using that technology
- [ ] Multiple filters can be active simultaneously (AND logic)
- [ ] Filter state managed with React state or URL params
- [ ] Smooth animation when projects filter in/out
- [ ] "Clear filters" button resets to show all projects
- [ ] Filter UI is responsive and accessible
