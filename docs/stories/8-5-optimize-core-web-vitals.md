## User Story

**As a** developer,  
**I want to** optimize Core Web Vitals,  
**so that** the portfolio meets Google's performance standards and provides excellent UX.

---

## Acceptance Criteria

- [ ] Largest Contentful Paint (LCP) < 2.5s on both mobile and desktop
- [ ] First Input Delay (FID) < 100ms
- [ ] Cumulative Layout Shift (CLS) < 0.1
- [ ] Font loading optimized with font-display: swap and preloading
- [ ] Critical CSS inlined for above-the-fold content
- [ ] Images use proper sizing to prevent layout shift
- [ ] Core Web Vitals validated using PageSpeed Insights and Chrome DevTools
