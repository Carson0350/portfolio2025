## Story Point Scale

- **1 point:** < 2 hours (simple configuration, small component)
- **2 points:** 2-4 hours (medium component, API integration)
- **3 points:** 4-8 hours (complex component, multiple integrations)
- **5 points:** 1-2 days (major feature, requires research)
- **8 points:** 2-3 days (epic-level work, break down further)

---

## Definition of Done

A story is "Done" when:

- [ ] Code implemented and passes TypeScript compilation
- [ ] All acceptance criteria met
- [ ] Code passes ESLint and Prettier checks
- [ ] Component/feature tested manually
- [ ] Responsive design verified (mobile, tablet, desktop)
- [ ] Accessibility checked (keyboard nav, screen reader)
- [ ] Code committed with conventional commit message
- [ ] Deployed to Vercel preview (if applicable)
- [ ] Reviewed against UI/UX spec and architecture
