## User Story

**As a** developer,  
**I want to** test all portfolio features end-to-end,  
**so that** everything works correctly before launch.

---

## Acceptance Criteria

- [ ] Navigation: All links work, smooth scrolling functions, active states update
- [ ] Hero section: CTA button scrolls to contact, animations play correctly
- [ ] About section: Bio content renders, hobbies display properly
- [ ] Skills section: All skills display with correct categories
- [ ] Experience section: Timeline renders, all jobs show correctly
- [ ] Projects section: Cards display, detail view opens/closes, filtering works (if implemented)
- [ ] Contact form: Validation works, submission succeeds, email received, rate limiting functions
